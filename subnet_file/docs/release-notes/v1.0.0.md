# Subnet v1.0.0

Subnet is a local social network that works inside shared WiFi networks.

## Features

- Discover nearby Subnet users on the same WiFi
- Create and host Spaces locally
- Join discovered Spaces over LAN
- Send live messages through TCP sockets
- Use Ghost Mode for anonymous participation
- Report users from message actions
- Customize accent color and text size
- Premium dark Material 3 interface

## Requirements

- Android device
- Same WiFi network for discovery and messaging
- Network/location permissions granted for best WiFi discovery behavior

## Installation

Download the APK from the release assets below and install it manually.

If Android blocks installation, allow installs from your browser or file manager
in system settings.

## Notes

Subnet is LAN-only. Some public, school, office, or campus WiFi networks block
multicast discovery or device-to-device traffic, which can prevent Spaces from
appearing even when both devices are connected to the same WiFi.
